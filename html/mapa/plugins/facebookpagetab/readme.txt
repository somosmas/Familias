== Installation ==
1. Copy the entire /facebookpagetab/ directory into your /plugins/ directory.
2. Activate the plugin.
3. Go to https://developers.facebook.com/apps
4. Press "Create new Application"
5. Fill in the name and add a small icon (PNG or GIF. Not ICO format)
6. Select Page Tab and fill in the addres of this view - http://your_domain.com/facebookpagetab/ (in case you have installed Ushahidi in a sub folder, there'll be additional stuff before /facebookpagetab/
